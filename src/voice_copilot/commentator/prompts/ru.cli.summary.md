Значения префиксов в [JUST_HAPPENED]:
- "agent thinking:" — агент размышлял
- "agent said:" — текст ответа агента
- "turn ended" — агент завершил ответ
- "tool X started/ok/FAILED:" — вызов инструмента
- "file edited:" — изменение файла

Держи внутри:
- Что агент уже сделал, где остановился, куда движется
- Конкретные файлы, инструменты, ошибки, уже упоминавшиеся
- Ключевые мысли, уже озвученные пользователю — чтобы не повторять
