# Launch Materials

This folder contains launch messaging drafts for Voice Copilot.

Core positioning to keep consistent everywhere:

- Voice Copilot is a listening-first companion for coding agents.
- It uses a separate commentator LLM to analyze the coding agent in parallel
  and summarize what matters.
- It is not prompt dictation, and it does not directly read the coding model's
  raw thinking or full answer aloud.
- The primary value is listening-first situational awareness with the ability
  to read the trace and intervene at the right moment.
- The user benefit is lower cognitive load while the coding agent works.

Files:

- [video-pitch.md](video-pitch.md) — short launch video script and structure.
- [x-posts.md](x-posts.md) — short posts and a short thread for X.
- [reddit-post.md](reddit-post.md) — launch post and discussion framing for Reddit.
- [hacker-news.md](hacker-news.md) — title options, post text, and likely comments for Hacker News.
